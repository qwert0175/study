# B형 설명회

### 등급구분

1. Intermediate(IM) : 반복문과 제어문으로 문제를 풀 수 있냐
2. Advanced(A) : Trial and Error에서 벗어나 
  - 완전탐색이나 시뮬레이션
  - 직관이 아닌 논리로 문제를 해결하는가
3. Profeddional(B) : 자료구조, 알고리즘 지식 학습 필요
  - 창의성 : 기존에 있는 것들의 조합 
4. Expert(C) : B형과 C형이 같이 시험을 쳐서 C를 넘으면 합격
