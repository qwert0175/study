### 왜 형변환을 해야하는가?

- 형변환을 자동으로 해주지 않는 경우
  
  ```
  # int + str
  print(1+'1')  # TypeError: unsupported operand type(s) for +: 'int' and 'str'
  ```
  
- 형변환을 자동으로 해주는 경우
  
  ```
  # int + float
  print(1+0.1)  # 1.1
  ```

- 정적타입 : 형변환을 해주지 않는 언어, 타입 지정해줘야 됨

- 동적타입 : 형변환을 해주는 언어, 타입 지정을 안해도 됨

- 파이썬은 동적타입 언어

- 쓰는 언어가 어떤 언어인지 알고 사용

### 함수

- 사용이유 : 재사용성, 가독성, 유지보수성

- 함수는 정의문과 호출문의 쌍으로 이루어짐

  ```
  # 함수 정의문
  def getSum(num1,num2):
    return num1 + num2

  # 함수 호출문
  getSum(1,2)
  ```

- 호출을 안하면 동작을 안함

### 매개변수(parameter) / 전달인자(argument)

- 매개변수 : 함수가 받을 값을 정의한 것

- 전달인자 : 함수가 실제로 받는 값

- 위치

  ```
  # 함수 정의문
  def getSum(**parameter**):
    return num1 + num2

  # 함수 호출문
  getSum(**argument**)
  ```

### 함수 호출

- 값 : 메모리에 들어있는 한 값, 여러개를 반환할 경우 자료구조형으로 하나만 반환

- 위치인자 : default, 위치를 통해 어떤 값이 어디에 들어가는지 확인

- 기본인자 : 매개변수에서 값을 지정, 함수사용 시 인자를 넣어주지 않으면 지정된 값을, 새 인자가 들어오면 해당 인자를 사용함

- 키워드인자 : 인자에서 매개변수의 값을 지정, 어떤 매개변수에 어떤 값을 넣을지 지정하므로 순서에 상관없음, 단, 키워드인자는 위치인자 뒤에 위치해야 함

### Scope

- 유효 범위(생명 주기)

- 함수는 호출되었을때부터 실행됨

  ```
  a = 1  # 전역변수 : 프로그램 끝날때까지 유지

  def test():
    b = 2  # 지역변수 : 함수가 끝날때까지 유지
    return b + 1

  c = test()
  ```

### 이름 검색 규칙

- LEGB Rule 예시

```
a = 1  # 1
b = 2  # 2

def enclosed():  #3
  a = 10  # 5
  c = 3  # 6

  def local(c):  # 7
    print(a, b, c)  # 9 print(10,2,500)
  
  local(500)  # 8
  print(a, b, c)  # 10 print(10,2,3)

enclosed()  # 4
print(a, b)  # 11 print(1,2)
```
