### 얕은 복사와 깊은 복사

- 예시

  ```
  a = 1
  b = a

  print(a,b) # 1 1

  a = 2

  print(a,b) # 2 1

  a = {'key' : 1}
  b = a

  print(a,b) # {'key' : 1} {'key' : 1}

  a['key'] = 2

  print(a,b) # {'key' : 2} {'key' : 2}
  ```

- 복사 시 자료형 구조는 메모리 주소를 복사

- 그래서 복사 후 자료형을 변경하면 복사된 것도 변경됨(얕은 복사로 인한 오류)
