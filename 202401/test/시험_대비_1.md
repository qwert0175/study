- enumerate는 파이썬 내장 함수 중 하나, 순서가 있는(iterable) 자료형(리스트, 튜플, 문자열 등)을 입력으로 받아 인덱스와 값을 포함하는 enumerate 객체를 반환

- enumerate(iterable, start=n) : 반복가능한 객체의 값을 n부터 시작하는 인덱스와 함께 enumerate 객체로 반환

9. enumerate 사용

```
# 아래의 코드를 실행했을 때, 결과를 예측하세요.
fruits = ['apple', 'banana', 'cherry']
result = ""

for index, fruit in enumerate(fruits):
    result += f"{index}. {fruit} | "

# rstrip() : 오른쪽 끝에서 지정한 문자를 제거   
result = result.rstrip(" | ")
print(result)

# 0. apple | 1. banana | 2. cherry
```

```
# 아래의 코드를 실행했을 때, 결과를 예측하세요.
animals = ['dog', 'cat', 'elephant', 'lion']
filtered_result = ""

for index, animal in enumerate(animals, start=1):
    if len(animal) > 3:
        filtered_result += f"{index}. {animal} | "

filtered_result = filtered_result.rstrip(" | ")
print(filtered_result)

# 3. elephant | 4. lion
```

```
# 아래의 코드를 실행했을 때, 결과를 예측하세요.
cities = ['Seoul', 'Tokyo', 'New York', 'Paris', 'Beijing']
filtered_result = ""

for index, city in enumerate(cities, start=1):
    if len(city) % 2 == 0:
        # lower() : 소문자로 변환하는 함수
        filtered_result += f"{index}. {city.lower()[::-1]} | "

filtered_result = filtered_result.rstrip(" | ")
print(filtered_result)

# 3. kroy wen
```

17. 재귀함수

```
# 주어진 함수를 사용하여 아래의 코드를 실행했을 때, 결과를 예측하세요.
def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)

print(factorial(5))
# 120
```

```
# 주어진 함수를 사용하여 아래의 코드를 실행했을 때, 결과를 예측하세요.
def fibonacci(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(6))
# 8
```

```

    else:
        return n + sum_from_1_to_n(n - 1)

# 10
```

30. Scope 문제

```
# 아래 코드의 결과를 예측해보세요.

def func1():
    x = 5
    return x

```

```
# 아래 코드의 결과를 예측해보세요.
a = 10
b = 20

def outer_func():
    global a
    a = 5
    b = 15

    def inner_func():
        return a + b

    return inner_func()

result = outer_func() + a + b
print(result)  # 45
```
