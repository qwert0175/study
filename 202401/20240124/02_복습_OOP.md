# 복습 OOP

<br>

### 객체 지향 프로그래밍

- 객체 : 객관적인 실체, 속성(데이터)과 행동(데이터를 수정)으로 이루어짐

- 지향 : Oriented => 기반

- 프로그래밍 : Program(순서도) + -ing(하는 것)

<br>

### 메서드

- 객체 안의 함수

- value가 함수인 것

<br>

### 클래스

- 객체를 생성하는 틀, 설계도

- 객체를 생성할 때 안의 값만 넣어주면 객체가 나옴

<br>

### 인스턴스

- 객체의 한 종류지만 객체와 인스턴스는 같지 않음

- 클래스에 의해 생성된 객체가 인스턴스

<br>

### 생성자 함수

- 자식을 생성하는 함수

- 타 언어에서는 클래스와 동일

- Python에서는 \_\_init__
