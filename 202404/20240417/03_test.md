### for문 에러

```

    console.log(array[i])

```

### null과 undefined의 차이

- null : 명시적 부재, 의도한 비어있음

- undefined : 암묵적 부재, javascript의 배려

### 잘 실행되는거 고르기

```
const ssafy = 1
const ssafy = 2
```

```
const ssafy
```

```
let ssafy
```

```
let ssafy = 1
let ssafy = 2
```

### 결과

```
const numbers = [1,2,3]
for (const number in numbers) {
    console.log(number)
}
```
