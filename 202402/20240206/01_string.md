### 패턴매칭에 사용되는 알고리즘

- 고지식한 패턴 검색 알고리즘

- 카프-라빈 알고리즘

- KMP 알고리즘

- 보이어-무어 알고리즘

### Brute Force 알고리즘

- 본문 문자열을 처음부터 끝까지 차례대로 순회하면서 일일이 비교하는 방식으로 동작

### KMP 알고리즘

- 불일치가 발생한 텍스트 스트링의 앞 부분에 어떤 문자가 있는지를 미리 알고 있으므로 불일치가 발생한 앞 부분에 대하여 다시 비교하지 않고 매칭을 수행

- 패턴을 전처리하여 배열 next[M]을 구해서 잘못된 시작을 최소화

- 시간복잡도 : O(M+N)

### 보이어-무어 알고리즘

- 패턴의 오른쪽부터 비교

- 대부분의 상용 소프트웨어에서 채택

### 문자열 암호화

- 시저 암호

- 문자 변환표를 이용한 암호화(단일 치환 암호)

- bit열의 암호화
